## Features
- Counting
- e926/e621
- Image Editing ([caption](#caption), speech bubble, etc...)
- [Confession System](#confession-system)
- Moderation
    * Ban Sync
    * Ticket System
    * Warn System (with Dashboard interface)
    * Server Logging
        - Git Diff is generated for message edits!
- Reminders
- Level System
